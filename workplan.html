Simple Risk Assessment

RISK ASSESSMENT

Risk #1

 The Development of the system likely will be slowed because project team members have not strong knowledge in PHP prior to this project.

Likelihood Risk

 High Probability of Risk

Potential impact on the project:

 This risk likely will increase the time to complete programming tasks by 50%.

Ways to Address this Risk

 It is very important that time and resources are allocated to up-front training in PHP for the programmers who are used for this project. Adequate training will reduce the initial learning curve for PHP when programming begins. Additionally, outside PHP expertise should be brought in for at least some part of the early programming tasks. This person should be used to provide experiential knowledge to the project team so that PHP-related issues are overcome

Functional Requirements:

1. Search and Browse

1.1 The system will allow alumni to browse current Announcements.

1.2 The system will allow alumni to search for the past and current Announcements.

1.3 The system will allow other Alumni to other Alumni basic information.

2. Communication

2.1 The system will keep the alumni updated and well informed.

2.2 The system will enable alumni to select and read the current or past Announcements of the school.

2.3 The system will notify the users via email for new Upcoming events.

3.Events Management

3.1 The System will enable the Admin to post new Announcements.

3.2 The System will allow Admin to accept pending comments.

3.3 The System will allow Admin to remove pending comments.

3.4 The System will allow Admin to remove Announcements.

4. User Management

4.1 The system will allow alumni to Edit Personal information

4.2 The system will allow alumni to Register

Non-Functional Requirements:

1. Operate

1.1 The system will run on any Web browsers.

1.2 The system will run on any Devices.

2. Performance

2.1 The browsing speed will kept at an acceptable level.

3. Security

3.1 Alumni personal information will be secured and keep private.

4. Cultural and Political

No special cultural and political requirements are expected.
